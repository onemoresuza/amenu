# amenu: Emulates Dmenu behaviors using default terminal

# Dependencies

-  [fzf](https://github.com/junegunn/fzf)
-  find & awk

## Installation

```sh
git clone https://github.com/salman-abedin/amenu.git && cd amenu && sudo make install
```

## Uninstallation

```sh
sudo make uninstall
```

- st
